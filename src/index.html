Restore Remix Backup Zip

Restore your Remix Zip backups to a workspace.
This only applies to zip files you created by using the 'Download all files button' on the homepage before you migrated Remix to use workspaces.
If your zip file contains a .workspaces folder this will be ignored, only files not in .workspaces will be imported.

1. Choose a zip file



2. Check your files down below
Scroll down and check the list of files that will be imported.

3. Make sure you have a workspace opened you want the files to be in.
Create a new workspace in the File Explorers and remove the demo files.

4. Set the root folder to ignore, ie remixbackup.
Your zip file probably has a root folder, called remixbackup, this import will ignore that and just import what is contained in it.



5. Import your files.

Import All Files


Or select individual files in the list below to import.
